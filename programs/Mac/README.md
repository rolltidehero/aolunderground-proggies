# Mac Hells

In the Mac AOL underground, hacking tools were called **"hells"** — the Mac equivalent of Windows "proggies." These included macro-based tools (KeyQuencer, OneClick, and RzaHell), 68k binary patchers, and other tools that exploited AOL's FDO protocol, client-trusted billing, and other design flaws in the Mac AOL client.

## Hells

| Hell | Author | Year | Description |
|------|--------|------|-------------|
| [AOL4Free v4](hells/aol4free/) | Happy Hardcore | 1995 | 68k binary patcher for AOL 2.6 — exploited client-trusted billing to access all paid services for free. Includes the original .sit archive, full 68k disassembly, 9 architecture diagrams, FDO protocol docs, and Happy Hardcore's own writings. |
